## テンプレートマッチングを使った画像認識サンプルプログラム

### 実行
```sh
# テンプレートマッチングでボールペンを検出
python template_matching1.py

# テンプレートマッチングでマウスを検出
python template_matching2.py

# テンプレートマッチングで隠れているマウスを検出
python template_matching3.py

# テンプレートマッチングで向きが違うマウスを検出
python template_matching4.py

# テンプレートマッチングでチェック柄でボールペンを検出
python template_matching5.py

# テンプレートマッチングでチェック柄で向きが違うボールペンを検出
python template_matching6.py
```
